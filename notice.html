작업 및 이용 안내

  • 작업 범위 Warudo API 세팅, 커스텀 API 개발, 간단한 3D 모델 제작을 진행합니다.
    • Warudo API 설치 및 커스텀 트리거 제작
    • 3D 모델 제작 및 Warudo 연동
    • Warudo PRO / 일반 버전 모두 작업
  • 세팅 방식 작업 난이도와 숙련도에 따라 아래 두 가지 방식으로 진행합니다. 세팅 방식에 따른 추가금은 발생하지 않습니다.
    • 원격 세팅 (초보자) — TeamViewer 또는 Google Chrome 원격으로 직접 세팅해 드립니다.
    • 파일 전달 (숙련자) — 설정 파일만 전달해 드리며, 파일 세팅 방법 안내에 따라 직접 적용하시는 방식입니다.
  • TeamViewer 설치 (원격 세팅용) 원격 세팅 전, 아래 순서대로 TeamViewer를 설치해 주세요.
    1. TeamViewer Windows 다운로드 페이지 접속
    2. 「TeamViewer」 → 「지금 다운로드」 클릭 후 설치 파일 실행
    3. 설치 중 「개인 / 비상업적 사용」 선택
    4. 설치 완료 후 TeamViewer 실행
    5. 화면에 표시되는 ID와 비밀번호 확인 비밀번호는 일정 시간마다 바뀔 수 있습니다. 세팅 직전에 다시 확인해 주세요.
    6. Discord(@im_rabbi)로 ID와 비밀번호를 보내 주시면 원격 세팅을 진행합니다.
  • Google Chrome 원격 데스크톱 설치 (원격 세팅용) TeamViewer 대신 Chrome 원격을 사용하실 경우, 아래 순서대로 설치해 주세요.
    1. Google Chrome 설치 (이미 설치되어 있다면 생략)
    2. Chrome에서 Chrome 원격 데스크톱 페이지 접속
    3. 「원격 액세스」 탭 선택
    4. 「다운로드」 → 「Chrome Remote Desktop 설치」 클릭
    5. 「원격 액세스 설정」 → 「시작하기」 클릭
    6. Google 계정으로 로그인
    7. 컴퓨터 이름 입력 (예: 방송용 PC)
    8. 6자리 PIN 번호 설정 (원격 접속용 비밀번호)
    9. 설정 완료 후 「온라인」 상태인지 확인
    10. Discord(@im_rabbi)로 Chrome 원격 사용 의사와 PIN 안내 요청을 보내 주세요.

    ※ Chrome 원격은 Google 계정 로그인이 필요합니다. 세팅 당일 PC가 켜져 있고 인터넷에 연결되어 있어야 합니다.

  • 가격 안내 작업 범위와 난이도에 따라 개별 견적을 안내해 드립니다.
    • 기본 항목은 가격표 페이지에서 확인할 수 있습니다.
    • 정확한 견적은 Discord(@im_rabbi)로 연락 부탁드립니다.

    • 비공개를 원하시는 경우 주문 전 미리 말씀해 주세요.
