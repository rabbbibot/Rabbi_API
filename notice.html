안내사항

작업 및 이용 안내

  • 작업 범위 Warudo API 연동, 팬 캐릭터 제작, 방송 연출 자동화 등을 진행합니다.
    • Warudo API 설정 및 커스텀 트리거 구성
    • 팬 캐릭터 Live2D / 3D 모델 제작 및 연동
    • OBS, 외부 프로그램과의 API 연동 지원
  • 세팅 방식 작업 난이도와 숙련도에 따라 아래 두 가지 방식으로 진행합니다.
    • 원격 세팅 (초보자) — TeamViewer 또는 Google Chrome 원격으로 직접 세팅해드립니다.
    • 파일 전달 (숙련자) — 설정 파일만 전달드리며, 직접 적용하시는 방식입니다.
  • TeamViewer 설치 (원격 세팅용) 원격 세팅 전, 아래 순서대로 TeamViewer를 설치해주세요.
    • TeamViewer Windows 다운로드 페이지 접속
    • 「TeamViewer」 → 「지금 다운로드」 클릭 후 설치 파일 실행
    • 설치 중 「개인 / 비상업적 사용」 선택
    • 설치 완료 후 TeamViewer 실행
    • 화면에 표시되는 ID와 비밀번호 확인
      • 비밀번호는 일정 시간마다 바뀔 수 있습니다. 세팅 직전에 다시 확인해주세요.
    • Discord(@im_rabbi)로 ID와 비밀번호를 보내주시면 원격 세팅을 진행합니다.
  • Google Chrome 원격 데스크톱 설치 (원격 세팅용) TeamViewer 대신 Chrome 원격을 사용하실 경우, 아래 순서대로 설치해주세요.
    • Google Chrome 브라우저가 없다면 먼저 설치
    • Chrome 원격 데스크톱 페이지 접속
    • 「원격 액세스」 탭 → 「다운로드」 클릭 (Chrome Remote Desktop 설치)
    • 「내 컴퓨터 설정」 또는 「설정」 클릭
    • 컴퓨터 이름 입력 (예: 방송용 PC)
    • 6자리 PIN 번호 설정 (원격 접속용 비밀번호)
    • 설정 완료 후 「온라인」 상태인지 확인
    • Discord(@im_rabbi)로 Chrome 원격 사용 의사와 PIN 안내 요청을 보내주세요.

    ※ Chrome 원격은 Google 계정 로그인이 필요합니다. 세팅 당일 PC가 켜져 있고 인터넷에 연결되어 있어야 합니다.

  • 가격 안내 고정 가격표는 별도로 운영하지 않습니다.
    • 작업 범위와 난이도에 따라 개별 견적을 안내드립니다.
    • 가격 문의는 Discord(@im_rabbi)로 연락 부탁드립니다.
  • Warudo 설치 Warudo는 Steam에서 설치할 수 있습니다.
    • Warudo Steam 페이지
    • API 기능 사용 시 Warudo Pro 라이선스가 필요할 수 있습니다.
  • API 연동 안내 Warudo API는 HTTP/WebSocket을 통해 외부 프로그램과 통신합니다.
    • API 포트 및 인증 토큰은 Warudo 설정에서 확인할 수 있습니다.
    • 방화벽에서 해당 포트가 허용되어 있는지 확인해주세요.
    • 자세한 설정 방법은 API 연동 페이지를 참고해주세요.
  • 문의 작업 문의 및 API 설정 도움이 필요하시면 연락 부탁드립니다.
    • Discord / Email 연락처는 소개 페이지에서 확인할 수 있습니다.
